 海外社媒热榜周报输出模板

## 标题格式

不加 `#` 符号，直接输出纯文本：

```
海外社媒热榜 YYYY年MM月DD日 - MM月DD日


## 板块结构

```
🎵 TikTok 热榜
🐦 Twitter / X 热搜
👾 Reddit 热帖 & 梗
📸 Instagram 趋势
🌐 跨平台热点（可选，内容在多平台均有大规模传播时启用）
```

## 每条内容格式

```
N. 内容名称（英文梗附中文释义）
   · 平台 & 数据：来源平台 + 核心量化数据（播放量/转推数/upvote等）
   · 走红原因：一句话说清楚为什么火
   · 传播形态：模板化二创 / 对口型配音 / 改编挑战 / 表情包 / 讨论话题 等
   · 关联内容：同系列梗 / 衍生话题（可选）
   🔗 直链（视频直链 / 原帖链接 / KnowYourMeme 词条）
```

链接规则：
- 有直链则附（TikTok 视频直链、Reddit 原帖、Twitter 推文）
- 无直链不放搜索入口，留空
- KnowYourMeme 词条链接优先于搜索结果页
- **禁止放排行榜/聚合页链接**，如 trends24.in、Google Trends、各类榜单首页——这类链接内容随时变化，对读者无参考价值。无合适直链则留空

## 板块顺序（固定）

1. 🌐 跨平台热点（有则优先放第一）
2. 🎵 TikTok 热榜
3. 🐦 Twitter / X 热搜
4. 👾 Reddit 热帖 & 梗
5. 📸 Instagram 趋势
6. 🎮 游戏官号可蹭热点速查（每期必出，放最后）

## 收录数量参考

| 板块 | 建议条数 |
|------|---------|
| 跨平台热点 | 0-2 条（有则附，无则不强行设此板块）|
| TikTok 热榜 | 3-5 条（音乐/模板/梗各至少 1 条）|
| Twitter/X 热搜 | 2-3 条 |
| Reddit 热帖 & 梗 | 2-4 条 |
| Instagram 趋势 | 1-2 条 |
| 游戏官号可蹭热点速查 | 3-5 条 |

## 游戏官号可蹭热点速查模板

从本期**已收录**的热点中筛选 3-5 个适合游戏官号跟拍或追梗的内容，每期必出，放在周报最后。

**强制规则：不得引入本期未收录的新内容。** 每条必须对应前面板块中已出现的某条热点，条目名称须与前文保持一致。

```
🎮 游戏官号可蹭热点速查（本周精选）

N. 热点名称（与前文一致）
   · 来源：🎵 TikTok / 🐦 Twitter / 👾 Reddit / 📸 Instagram / 🌐 跨平台
   · 参考方向：具体执行思路（素材怎么用 / 怎么嫁接游戏内容）
   · 理由：为什么适合游戏官号跟（受众重合 / 制作成本 / 互动预期）
```

筛选标准：
- 受众与游戏玩家重合度高（18-35岁、娱乐向、二次元/动漫/流行文化）
- 参与门槛低，不依赖外部 IP 版权（或可规避版权风险）
- 格式固定、易复刻，官号美术/运营团队能快速执行
- 优先选跨平台传播、有持续传播证据的热点，避免一日昙花
